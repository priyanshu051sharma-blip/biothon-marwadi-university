## 🎯 MAIN FEATURES EXPLAINED

### 1. Symptom Assessment
**How it works:**
1. User describes symptoms (e.g., "chest pain, breathlessness")
2. AI asks detailed follow-up questions
3. Asks about lifestyle (sleep, diet, stress)
4. Compiles comprehensive patient profile

**Output:** Complete symptom analysis with risk factors

---

### 2. Disease Prediction
**How it works:**
1. Uses Bayesian inference
2. Matches symptoms to diseases
3. Calculates confidence scores
4. Assigns urgency levels

**Diseases Covered:**
- Pneumonia (87% accuracy)
- COVID-19 (85% accuracy)
- Migraine (92% accuracy)
- Gastroenteritis (89% accuracy)
- Acute Coronary Syndrome (91% accuracy)
- Angina Pectoris (88% accuracy)
- Common Cold (84% accuracy)
- Diabetes (86% accuracy)
- Hypertension (87% accuracy)
- Anxiety Disorder (83% accuracy)

**Urgency Levels:**
- 🔴 EMERGENCY (call ambulance immediately)
- 🟠 URGENT (visit hospital today)
- 🟡 MODERATE (schedule appointment)
- 🟢 NON-URGENT (regular checkup)

---

### 3. Doctor Finder
**How it works:**
1. Identifies required specialization
2. Filters by location
3. Sorts by rating
4. Shows real-time availability

**Doctor Info:**
- Name, specialization, experience
- Hospital, location, ratings
- Consultation fees
- Languages spoken
- Available appointment slots
- Online consultation ready

**Example:**
- For pneumonia: Pulmonologists
- For chest pain: Cardiologists
- For headache: Neurologists
- General: Family Physicians

---

### 4. Appointment Booking
**How it works:**
1. Select doctor from recommendations
2. View available time slots
3. Choose preferred slot
4. Confirm booking
5. Get appointment ID
6. Receive online consultation link

**Features:**
- Instant confirmation
- SMS/Email notification ready
- Appointment reminders
- Cancellation with refund
- Follow-up scheduling

---

### 5. Medicine Finder
**How it works:**
1. Search by medicine name
2. See all brands available
3. Check pricing
4. Find nearby pharmacies
5. Check availability
6. Order online

**Medicine Info:**
- Generic & brand names
- Dosage options
- Price ranges
- Pharmacy locations
- Stock availability
- Prescription requirements
- Home delivery

**Example:**
- Paracetamol: ₹20-150, 5000+ units in stock
- Aspirin: ₹50-200, immediate availability
- Amoxicillin: ₹100-500, prescription required

---

### 6. Blood Bank Locator
**How it works:**
1. Enter blood type (O+, O-, A+, etc.)
2. Select location/city
3. See available units
4. Check emergency services
5. Reserve blood
6. Arrange delivery

**Features:**
- Real-time inventory
- 24/7 emergency service

- Quick reservation
- Cost ₹2500 per unit

**Locations:**
